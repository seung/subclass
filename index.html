The bare dancefloor is created when the page is loaded, where the types of dancers is passed to the 'makeDanceFloor' function and the dancefloor is, in turn, passed to the 'setupControls' function. Upon clicking a link to create a dancer, the attached jQuery function passes the dancer type to the 'makeDanceFloor.makeDancer' function. This function creates the dancer, appends/draws it to the screen and pushes the dancer to the 'window.dancers' array.

The base dancer is 'blinkyDancer'.

A poleDancer inherits from blinkyDancer and adds a CSS property of "height: 30px", and changed the 'border-color' to green and 'border-width' to 3px.

A coloredDancer inherits from blinkyDancer and changes the CSS property of 'border-color' to a randomly generated RGB value.

The mixin 'makeSquareDancer' loops through the dancer array and if the randomly generated number is greater than 0.9, the function changes the 'border-radius' of the currently selected dancer to 0px, thus making the border square.

emergentBehavior.js loops through all the currently displayed dancers and changes their position if they are within 10000px of any dancer that comes after them in the dancers array.
