The Program starts in the main file which constructs a stage and calls the controls setup which links the buttons to make dancer. When a make dancer is clicked the dancer is made by calling the appropriate makeDancer function and then use jQuery to append to the stage. The mixins do not add new dancers; they simply select a random dancer that has already been created and changes their properties.
