Bance Dattle!

Clicking one of the buttons tells the dance floor to create the appropriate dancer on the stage. All dancers are created at random locations.

  • Blinky Dancer:
      Uses a set interval to toggle visibility on and off.
  • Rave Dancer:
      Cycles through an array of colors ('#FF00CC','#F7FF00','#5EFF00','#00FFF2') using Math.random and array.length.
  • Mosh Pit Dancer:
      • Mosh pit dancers are created in a separate div. This div is given a random position every interval, and modulus is used to keep it on the screen. The individual dancers are also each given a random position, mod 100 to keep them inside the div.
