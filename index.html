So How Does This Thing Work, Anyway?

tl;dr:

When a user clicks on of the "make a dancer" buttons on this page, a new dancer should appear in the middle of the page.

Gruesome Details

When the page loads, main.js gets loaded and calls controls.js. controls.js sets up a listener for any HTML element with the class clickable. If a user clicks that element, and if that element has data within the data-dancer-creator property that matches a function name, it calls that function.
